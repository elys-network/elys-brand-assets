Expletus Sans Variable Font
===========================

This download contains Expletus Sans as both variable fonts and static fonts.

Expletus Sans is a variable font with this axis:
  wght

This means all the styles are contained in these files:
  ExpletusSans-VariableFont_wght.ttf
  ExpletusSans-Italic-VariableFont_wght.ttf

If your app fully supports variable fonts, you can now pick intermediate styles
that arenâ€™t available as static fonts. Not all apps support variable fonts, and
in those cases you can use the static font files for Expletus Sans:
  static/ExpletusSans-Regular.ttf
  static/ExpletusSans-Medium.ttf
  static/ExpletusSans-SemiBold.ttf
  static/ExpletusSans-Bold.ttf
  static/ExpletusSans-Italic.ttf
  static/ExpletusSans-MediumItalic.ttf
  static/ExpletusSans-SemiBoldItalic.ttf
  static/ExpletusSans-BoldItalic.ttf

Getting Started
---------------

1. Installation: Begin by installing the font files of your choice.
2. Font Selection: Utilize your application's font picker to explore the available styles within the Expletus Sans font family.
